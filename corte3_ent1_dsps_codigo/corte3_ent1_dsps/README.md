# Corte 3 - Entrega 1 - DSPS

Soluciones en C++ para:

- LeetCode 1396 - Design Underground System
- LeetCode 2241 - Design an ATM Machine *(en la asignacion aparece como 3709, pero el enunciado publico corresponde a 2241)*
- LeetCode 3815 - Design Auction System
- Exercism Meetup in C++
- Exercism Clock in C++

## Compilacion

```bash
mkdir build
cd build
cmake ..
cmake --build .
```

## Ejecucion

Desde `build`:

```bash
./leetcode_1396_underground
./leetcode_2241_atm
./leetcode_3815_auction
./exercism_meetup
./exercism_clock
```
